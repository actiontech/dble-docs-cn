<h1>server.xml 配置</h1>

<h2>整体XML结构</h2>

+ system (目前reload不会做变更)
  - properties 
+ user (可多值,reload可变更)
+ firewall (reload可变更)
  - whitehost
  - blacklist
  - host
+ alarm (reload可变更)
