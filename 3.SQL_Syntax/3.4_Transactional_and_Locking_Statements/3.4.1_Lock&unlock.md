###  3.4.1 Lock&unlock

####  3.4.1.1  Syntax
LOCK TABLES   tbl_name [[AS] alias]  <b>lock_type</b>

<b>lock_type</b>:  READ |  WRITE

<br/>

UNLOCK TABLES

####  3.4.1.2 举例

```
lock tables test_table read;
unlock tables;

```

####  3.4.1.3 限制

1. 当前session加锁后访问其他表可能不会被阻止或者报错。     
2. 加写锁后，复杂查询可能不会返回正确结果。
