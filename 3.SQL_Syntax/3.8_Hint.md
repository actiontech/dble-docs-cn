### 3.8.2 举例

```
/*!dble:sql=select 1 from sbtest */ call p_show_time();
/*!dble:schema=testdb */ call p_show_time();
/*!dble:datanode=dn1*/ update sbtest set name = 'test';
/*!dble:db_type=master*/ select count(*) from sbtest;
/*#dble:sql=select 1 from sbtest */ call p_show_time();
/*#dble:schema=testdb */ call p_show_time();
/*#dble:datanode=dn1*/ update sbtest set name = 'test';
/*#dble:db_type=master*/ select count(*) from sbtest;
```
