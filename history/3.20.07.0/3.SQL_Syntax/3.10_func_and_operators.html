3.10 函数与操作符支持列表(alpha版本)

3.10 函数与操作符支持列表(alpha版本)

3.10.0 注意：

  1. 如果能保证SQL会将函数整体下发给某个后端结点，函数支持度没有意义，支持度将托管于MySQL.
  2. 在2.18.09.0 版本之前（含），涉及到中文字符集的字符串相关函数有bug。
  3. 除了聚合函数经过充分测试之外，其余函数只进行了简单的测试，并没有全覆盖测试，请使用前充分测试 。
  4. 未在列表中的函数，一概不支持。
