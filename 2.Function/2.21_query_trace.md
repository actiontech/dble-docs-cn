## 2.21 单条SQL性能trace

这个功能与MySQL的profile功能类似，用于统计一条查询在dble内部的各个阶段的耗时情况，用来发现性能瓶颈，改进SQL或者dble配置，达到提高性能的目的。
此功能为session 级别。  

**1.  可以通过此命令检查trace功能是否开启**     

```
mysql> select @@trace;
+---------+
| @@trace |
+---------+
| 0       |
+---------+
1 row in set (0.02 sec)
```
**2.  开启 trace 功能**
```
mysql> set trace =1;
Query OK, 0 rows affected (0.09 sec)

mysql> select @@trace;
+---------+
| @@trace |
+---------+
| 1       |
+---------+
1 row in set (0.00 sec)
```
