### 2.1.1  select 命令

#### 2.1.1.1  select @@VERSION_COMMENT
##### select @@VERSION_COMMENT；
描述：查询dble的版本信息；  
例：  
  
```
MySQL [(none)]> select @@VERSION_COMMENT;
+--------------------------+
| @@VERSION_COMMENT        |
+--------------------------+
| dble Server (ActionTech) |
+--------------------------+
1 row in set (0.02 sec)
```  


列描述：  
略
