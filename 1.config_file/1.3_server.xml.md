
## 1.3 server.xml 配置

### 整体XML结构

+ system (目前reload不会做变更)  
    - properties   
+ user (可多值,reload可变更)
+ firewall (reload可变更)
    - whitehost
    - blacklist
    - host
	
### 配置选项

##### 注意:"实例/全局属性" 指的是如果集群配置多个中间件,是否要求配置在全局级别保持一致，或者实例间可以不同 

### version

在2.18.12.0之后的版本中server.xml在文件头中添加了version属性，以供运维人员区分配置创建或修改的版本
